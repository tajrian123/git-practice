# Git & GitHub Practice Lab

## Summary

This repository contains the work completed for the **Git & GitHub Practice** laboratory exercise. The objective of this lab was to gain practical experience with Git and GitHub by creating a public repository, cloning it to a local machine, modifying a C++ program with personal information, tracking changes, committing updates, and pushing them to GitHub. This exercise helped me understand the basic Git workflow and the importance of version control in software development.

---

## Overview

This repository demonstrates the fundamental concepts of Git and GitHub through a simple C++ application. The program stores and displays a student's personal and academic information, including name, student ID, section, department, and university. The repository was used to practice common Git operations such as cloning, editing files, staging changes, committing updates, and synchronizing with a remote GitHub repository.

## Objectives

* Understand the fundamentals of Git and GitHub.
* Create and manage a public GitHub repository.
* Clone a remote repository to a local machine.
* Modify and update project files.
* Track changes using Git.
* Stage and commit changes with meaningful commit messages.
* Push local commits to GitHub.
* Gain practical experience with the standard Git workflow.

## Tasks Performed

* Created a public GitHub repository.
* Added the initial `main.cpp` file.
* Cloned the repository to the local computer.
* Verified the repository status using Git.
* Updated the C++ program with personal information.
* Added additional fields such as Department and University.
* Viewed file changes using Git.
* Staged and committed the modified file.
* Pushed the updated repository to GitHub.

## Program Description

The C++ program performs the following tasks:

* Stores student information using variables.
* Stores academic information such as department and university.
* Displays all information in a well-formatted output.
* Prints a welcome message for the Git & GitHub Practice exercise.

## Git Commands Practiced

```bash
git clone <repository_url>
git status
git diff
git add main.cpp
git add .
git commit -m "Updated student information"
git push
```
